# Sluice — demo video script (< 3:00)

Record at 1440×900, dark theme, live site (https://sluiceflow.vercel.app). Cursor visible, no cuts
mid-payment — the point is that everything settles for real while the viewer watches.

| Time | Shot | Say |
| --- | --- | --- |
| 0:00 | **Landing hero** — the physical rail photo behind the copy, the animated meter below it. Hold 5s, slow scroll to the live stats strip. | "AI reads the web for free. Sluice is the open toll booth: any unit of value — a read, a second, a citation — metered and settled on Arc in USDC. Every number on this page is real." |
| 0:20 | **Ask the agent** (/ask) — type: "How does Circle Gateway settle nanopayments on Arc?" → answer arrives with inline receipts. Zoom on one receipt strip. | "Ask it a question and watch it pay every source it cites — for real. Each citation is a USDC micro-payment to that author, on Arc. The payment *is* the citation." |
| 1:00 | **Creator Studio** (/app/earn) — paste an RSS feed URL, ingest; show the citable source card with its RSL, llms.txt, and the live earned badge. | "Creators price their own work — a feed becomes citable sources in one click, with an earned counter driven by real settlements. Multi-author work splits on-chain via a RoyaltySplitter." |
| 1:30 | **Agent Console** (/app/spend) — open the R7 Journey Agent's session; scroll the decision trace: paid / skipped / capped with reasoning; point at the budget bar. | "Buyer agents show their work: the model scores relevance, but deterministic code enforces the budget, the price ceiling, and the allowed units. Model output never authorizes a payment. Here it paid four relevant sources and skipped the noise — including another team's x402 endpoint it paid at *their* price." |
| 2:00 | **Settlements** (/app/settlements) — the payments just made appear; click Verify on one → Circle transfer ID. Switch tab to Arcscan showing the BondEscrow/FundingPool contract page. | "Every payment leaves a receipt with its Circle transfer ID, and the on-chain anchors — deposits, withdrawals, bonds, funding sweeps — are open on Arcscan for anyone to check." |
| 2:30 | **Traction** (/traction) — the honest scoreboard: distinct humans, clustered wallets, cross-team partners strip. End on the tagline over the hero. | "Traction is counted conservatively — one profile is one human, linked wallets collapse into one. The web already knows how to declare a price. Sluice is how it gets paid." |

## Shot list / prep
- Pre-fund nothing — the faucet, deposits, and payments should happen on camera where timing allows.
- Have /ask, /app/earn, /app/spend, /app/settlements, /traction pinned as tabs; Arcscan contract page in a sixth tab.
- Reduced-motion OFF, browser zoom 100%, hide bookmarks bar.
- If a settlement is still "batching" on camera, say so — honest states are the feature.
